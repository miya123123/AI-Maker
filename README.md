# AI Maker

AI Makerは、自分でステージを作成して遊べる2Dプラットフォーマーゲームです。
Gemini API (Gemini 3 Flash) を活用し、テキストのプロンプトからAIにステージを自動生成させることも可能です。

## 機能

- **ステージエディター**: ブロック、トゲ、ジャンプ台、敵(縦)、敵(横)、コイン、鍵、セーブポイント、プレイヤー、ゴールを自由に配置してオリジナルのステージを作成できます。
- **AIステージ生成**: 「難しいステージ」「階段が多いステージ」など、テキストで指示を出すだけでAIがステージを自動生成します。
- **プレイモード**: 作成したステージを実際にプレイできます。

## プレイモードでの操作方法

- `←` `→` キー: 左右移動
- `↑` または `Z` キー: ジャンプ

## 起動方法
1. プロジェクトのルートディレクトリに `.env` ファイルを作成し、Gemini APIキーを設定してください。（`.env.example` が参考になります）
   ```env
   GEMINI_API_KEY="あなたのAPIキー"
   ```
2. プロジェクトのディレクトリで以下のコマンドを実行し、依存関係をインストールします。
   ```bash
   npm install
   ```
3. 以下のコマンドを実行して、開発サーバーを起動します。
   ```bash
   npm run dev
   ```
4. ターミナルに表示されるローカルURL（例: `http://localhost:3000/` など）をブラウザで開いてプレイします。
